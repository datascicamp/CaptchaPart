CAPTCHA Module Tutorial

First, use GET method query uri '/api/hash-match/captcha' to get a hash-CAPTCHA pair.

(P.S. the hash-CAPTCHA pair is maintained in Response.Headers.CAPTCHA and Response.Headers.Hash-Code)


Then, using the hash code you've been given query uri '/api/hash-match/captcha/< hash_code >', you will get an CAPTCHA image with the CAPTCHA code you already have.


Finally, DO NOT forget to remove the expired hash-CAPTCHA pair by using '/api/captcha/hash-code/< seconds-passed >' with DELETE method.
